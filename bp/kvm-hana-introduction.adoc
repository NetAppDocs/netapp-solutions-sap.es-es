= Implemente SAP HANA en SUSE KVM con almacenamiento NetApp mediante SR-IOV y NFS.
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Implemente SAP HANA Single-Host en SUSE KVM utilizando almacenamiento NetApp con interfaces de red SR-IOV y acceso a almacenamiento NFS o FCP.  Siga este flujo de trabajo para configurar las interfaces virtuales, asignarlas a las máquinas virtuales y configurar las conexiones de almacenamiento para un rendimiento óptimo.

Para obtener una descripción general de SAP HANA en la virtualización KVM, consulte la documentación de SUSE: https://documentation.suse.com/sbp/sap-15/pdf/SBP-SLES4SAP-HANAonKVM-SLES15SP5_en.pdf["Mejores prácticas de SUSE para SAP HANA en KVM"] .

image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["1"]link:kvm-hana-requirements.html["Revisar los requisitos de configuración"]

[role="quick-margin-para"]
Revisa los requisitos clave para implementar SAP HANA en SUSE KVM utilizando almacenamiento NetApp con SR-IOV y protocolos de almacenamiento.

image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dos"]link:kvm-hana-network.html["Configurar interfaces de red SR-IOV"]

[role="quick-margin-para"]
Configure SR-IOV (Virtualización de E/S de raíz única) en el host KVM y asigne interfaces virtuales a la máquina virtual para la comunicación de red y el acceso al almacenamiento.

image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Tres"]link:kvm-hana-fcn.html["Configurar redes Fibre Channel"]

[role="quick-margin-para"]
Asigne puertos HBA FCP físicos a la máquina virtual como dispositivos PCI para usar LUN FCP con SAP HANA.

image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Cuatro"]link:kvm-hana-storage.html["Configurar el almacenamiento de NetApp para SAP HANA"]

[role="quick-margin-para"]
Configure las conexiones de almacenamiento NFS o FCP entre la máquina virtual y los sistemas de almacenamiento NetApp para los archivos de la base de datos SAP HANA.
