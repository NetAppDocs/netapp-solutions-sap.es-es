= Recuperación ante desastres de SAP HANA con replicación de almacenamiento
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Este documento ofrece una visión general de las opciones de protección de recuperación ante desastres para SAP HANA. Incluye información detallada sobre la configuración y una descripción de un caso de uso de una solución de recuperación ante desastres en tres sitios basada en la replicación de almacenamiento síncrona y asíncrona de NetApp SnapMirror. La solución descrita utiliza NetApp SnapCenter con el plug-in de SAP HANA para gestionar la coherencia de la base de datos.

Autor: Nils Bauer, NetApp
