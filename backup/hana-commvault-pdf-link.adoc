= SAP HANA copia de seguridad y recuperación usando NetApp Storage Systems y Commvault Software
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Este documento describe el diseño de una solución de NetApp y Commvault para SAP HANA, que incluye la tecnología de gestión de instantáneas Commvault IntelliSnap y la tecnología de instantáneas NetApp. La solución se basa en el almacenamiento NetApp y la suite de protección de datos Commvault.

Autores: Marco Schoen, NetApp; Dr. Tristan Daude, CommVault Systems
