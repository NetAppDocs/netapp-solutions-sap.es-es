= Obtenga información sobre cómo configurar SnapCenter para entornos SAP HANA
:allow-uri-read: 


[role="lead"]
Configure SnapCenter para entornos SAP HANA mediante un enfoque de dos fases: configuración inicial para recursos compartidos (credenciales, sistemas de almacenamiento y políticas) y configuración específica de recursos para sistemas HANA individuales (implementación de host, descubrimiento automático y configuraciones de protección).

La configuración de SnapCenter para un entorno SAP HANA con múltiples sistemas HANA se puede dividir en dos áreas principales:

* La configuración inicial
+
** Configuraciones de credenciales, almacenamiento y políticas. + Estas configuraciones o recursos normalmente son consumidos por múltiples sistemas HANA.


* La configuración específica del recurso HANA
+
** La configuración de la protección del host, de HANA y de los recursos debe realizarse para cada sistema HANA individualmente.
