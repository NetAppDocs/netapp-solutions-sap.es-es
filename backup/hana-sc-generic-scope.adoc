= Proteja los sistemas SAP HANA con SnapCenter en ONTAP, Azure NetApp Files y FSx para ONTAP
:allow-uri-read: 


[role="lead"]
Proteja los sistemas SAP HANA con NetApp SnapCenter utilizando copias de seguridad basadas en instantáneas y replicación de datos. Esta solución cubre la configuración de SnapCenter y las mejores prácticas operativas para sistemas SAP HANA en sistemas ONTAP AFF y ASA , Azure NetApp Files y Amazon FSx para ONTAP, incluidas estrategias de respaldo, controles de coherencia y flujos de trabajo de recuperación.

Autor: Nils Bauer, NetApp

Se pueden encontrar detalles adicionales específicos de casos de uso sobre las operaciones de actualización del sistema SAP y la replicación del sistema SAP HANA en:
